## 🎯 Project Overview

The **Expense Tracker** is a Python-based application that helps users record, manage, and analyze their daily expenses.

It allows users to:
- Add expenses with categories  
- View all expenses  
- Generate summaries  
- Visualize spending patterns using graphs  

This project demonstrates real-world use of **data handling, analysis, and visualization**.

## 🎯 How It Works

1. User selects an option from the menu  
2. Adds expense (amount + category)  
3. Data is stored in `data.csv`  
4. User can view all expenses  
5. Summary is generated using Pandas  
6. Graph is displayed using Matplotlib
